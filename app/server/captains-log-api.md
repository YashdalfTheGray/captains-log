# Captain's Log REST API

This defines the REST API for the Captain's Log server.

All endpoints are mounted at `/api/v1`.

## Endpoints

### `GET /logs`

### `POST /logs`

### `PUT /logs/:id`

### `DELETE /logs/:id`

### `GET /reports/week`
This might possibly go into a module of it's own.

### `GET /reports/day`
This might possibly go into a module of it's own.
